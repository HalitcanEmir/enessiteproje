# Proje Görselleri

Bu klasöre hero slider'da kullanılacak 3 adet yüksek kaliteli görsel ekleyin.

## Görsel Gereksinimleri

- **Format:** JPG veya PNG
- **Boyut:** En az 1920x1080 (Full HD), tercihen 3840x2160 (4K)
- **Oran:** 16:9 veya benzer geniş ekran oranı
- **Dosya Adları:**
  - `project-1.jpg` - İlk proje görseli
  - `project-2.jpg` - İkinci proje görseli
  - `project-3.jpg` - Üçüncü proje görseli

## Görsel İçeriği

Görseller şu konularda olmalıdır:
- Elektrik-elektronik projeleri
- Taahhüt işleri
- Kurumsal tesisler
- Endüstriyel uygulamalar
- Saha çalışmaları

Görseller yatay (landscape) formatında ve yüksek kaliteli olmalıdır.

## Not

Görselleri ekledikten sonra projeyi yeniden başlatmanız gerekebilir:
```bash
npm run dev
```
