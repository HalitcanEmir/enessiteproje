# Referans Logoları

Bu klasöre referans şirket/kurum logolarını ekleyin.

## Görsel Gereksinimleri

- **Format:** PNG veya SVG (şeffaf arka plan tercih edilir)
- **Boyut:** En az 400x300px, tercihen 800x600px veya daha yüksek
- **Oran:** 4:3 veya benzer oran (dikey veya yatay)
- **Dosya Adları:**
  - `toki-logo.png` - TOKI logo
  - `ashb-logo.png` - Aile ve Sosyal Hizmetler Bakanlığı logo
  - `ticaret-logo.png` - Ticaret Bakanlığı logo
  - `meb-logo.png` - Milli Eğitim Bakanlığı logo
  - `saglik-logo.png` - Sağlık Bakanlığı logo
  - `ref6-logo.png` - Referans 6 logo
  - `ref7-logo.png` - Referans 7 logo
  - `ref8-logo.png` - Referans 8 logo
  - `ref9-logo.png` - Referans 9 logo
  - `ref10-logo.png` - Referans 10 logo

## Carousel Davranışı

- Her seferinde 6 logo yan yana gösterilir
- 2 saniye bekler, sonra yumuşak bir şekilde kayar
- Slider ilerledikçe: 1-6, 2-7, 3-8, 4-9, 5-10, 6-1 (loop) şeklinde döner

## Not

Logoları ekledikten sonra projeyi yeniden başlatmanız gerekebilir:
```bash
npm run dev
```
